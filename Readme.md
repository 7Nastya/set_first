# Фан-блог на Python 

Выполняет следующие функции:

1. Регистрация, вход пользователя
2. Добавление, изменение, удаление статей
3. Реализация пагинации страниц 
4. Есть возможность использования меню 
5. Комментирование статей пользователями(изменение, удаление)
   


## Инстурменты 

1. DJANGO
2. API
3. SQLITE 
4. CELERY, REDIS
5. DJANGO REST Framework
6. CSS, HTML

## Создании миграции
- python manage.py makemigrations
- python manage.py migrate
